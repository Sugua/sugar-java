    UML 类图
        类
            第一行：类名称
            第二行：特性（属性或字段）
                '+'表示public
                '-'表示private
                '#'表示protected
            第三行：操作（方法或行为）
            注意：若类名是斜体字，则此类为抽象类

        接口
            矩形表示法，顶端有<<interface>>
                第一行：接口名称
                第二行：接口方法
            棒棒糖表示法
